Markdown Formatting Guide

Use these custom extensions in your blog posts in addition to standard GitHub-Flavored Markdown.

CTA Button

Renders a styled call-to-action button link.

![CTA-BUTTON]{Button Text}[https://example.com]
Image with Caption

Renders a full-width image with an italic caption below it.

![IMAGE-Caption text here][https://example.com/image.png]
Image Carousel

Renders a horizontally scrollable carousel. Click links are optional (use empty {} to skip one). Alts and links must match image count.

!![IMAGE-CAR]{Alt for image 1}{Alt for image 2}[https://url1.com/img.png][https://url2.com/img.png]{https://optional-link1.com}{}
Alert / Callout Blocks

GitHub-style alert blocks. Supported types: INFO, WARNING, ERROR, NOTE, TIP, IMPORTANT, CAUTION.

> [!INFO]
> This is an info block.

> [!WARNING]
> This is a warning.
